# Scaffold — Reference File Shapes

These are the canonical starting shapes for a generated agent project. They are **copy-and-substitute
references**, not part of this boilerplate's own importable code.

## How the builder uses this

When scaffolding a new agent (see `.claude/agents/agent-builder.md` Stage 4–5), copy these files into
the repo root and substitute the placeholder token:

- **`<package>`** — your project's snake_case Python package name (matches the agent slug). It appears
  both in directory paths (`src/<package>/...`) and inside import statements. Replace every occurrence.

The directory layout these files assume is defined once in
[`../spec/engineering/project-layout.md`](../spec/engineering/project-layout.md) — keep them in sync.

## What's here

| Path | Purpose |
|------|---------|
| `src/<package>/config/settings.py` | Pydantic `BaseSettings` singleton with env prefix |
| `src/<package>/db/session.py` | engine + sessionmaker + `init_db` |
| `src/<package>/db/models.py` | SQLAlchemy 2.0 declarative base + `RunRow` |
| `src/<package>/graph/state.py` | `AgentState` TypedDict |
| `src/<package>/graph/nodes.py` | stub node functions `(state) → state` |
| `src/<package>/graph/edges.py` | conditional routing functions |
| `src/<package>/graph/agent.py` | `StateGraph` compiled once at startup |
| `src/<package>/graph/runner.py` | `run_agent()` entry point |
| `src/<package>/api/__init__.py` | `create_app()` factory + lifespan |
| `src/<package>/api/_common.py` | `ok()` / `api_error()` response envelope |
| `tests/conftest.py` | settings-singleton reset fixture |
| `tests/integration/test_pipeline.py` | stub run → one DB record → status `completed` |
| `alembic/script.py.mako` | **must exist before any alembic command** — not generated by anything |

For a **ReAct** agent (acts on the outside world via tools/data/search), the stub `nodes.py`/`state.py`
below are the one-shot starting point — extend them per
[`../spec/engineering/patterns/react-agent.md`](../spec/engineering/patterns/react-agent.md).

The default model in `settings.py` (`claude-sonnet-4-6`) is the source-of-truth default from
[`../spec/engineering/tech-stack.md`](../spec/engineering/tech-stack.md) § Models.
